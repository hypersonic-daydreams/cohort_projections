## Decision

**`promote`** — as modified: `m2026r1` / `cfg-20260611-production-lock`.

Per-decision disposition (ADR-061):

| Item | Verdict | Basis |
|---|---|---|
| D1 (25–29 smoothing extension) | **Accept** | Large clean-base college-county gains (GF −3.73, Cass −1.70, Ward −0.69) at moderate state cost |
| D2 (GQ fraction parameter) | **Accept; calibrate to 0.75** | EXP-C clean run: best state APE-short, rural and overall improvements; +3.0k state @2050 forward |
| D3 (extended convergence hold) | **Reject for this release** | Original under-projection justification was substantially a harness artifact; backtest-invisible; forward effect (+11.7k @2055) is a stance change not required by accuracy evidence. Revisit next cycle |
| D4 (12-county college list) | **Accept as modified — Williams removed** | Williams below ADR-061's own 2.5% threshold; inclusion worsened Williams +0.90pp (ADR-067 F3) |
| EXP-B (blend 0.7) | **Reject for production** | County gains cost state APE-medium 1.46→2.64 and bias +0.97→+1.64; headline metric is the state total. Recorded as next-cycle research (monotone-toward-county-pure pattern) |
| Pre-2026-06-11 pending bundles (EXP-A/E/I/J/K, 2026-05-27 search bundles) | **Superseded** | Contaminated rate base (ADR-067 F1); re-derive from raw-base matrix if needed |

Decision rationale: the public product is a county projection series whose
headline is the state total. The locked config takes the college-county
accuracy gains where public scrutiny is highest (Grand Forks, Cass, Ward),
keeps the state-level bias drift to the smallest among college-fix variants,
removes the one demonstrated mis-inclusion (Williams), and adopts the one
calibration with broad accuracy support (GQ 0.75). Continuity with the
2026-05-27 provisional baseline is preserved up to the Williams and GQ
changes (≈ +3–4k state @2050, quantified at the Stage-3 rerun).
