# How To Use The 2024 SDC PDF

Reference file: `data/raw/ND Population Projections.pdf`

The 2024 State Data Center PDF is the best starting point for the 2026 public
release because it has the right scale and public orientation. It is a compact
16-page report for the state, regions, and counties, with enough narrative to
stand alone and enough tables to be useful.

Use it as a design and pacing reference. Do not use it as a language template.
The 2026 release has a different model, a longer horizon, and a stronger
scenario frame.

## What To Keep

Keep the basic public-report shape:

- Cover and agency attribution.
- Table of contents.
- Executive summary.
- Statewide population chart.
- Short methodology section.
- Regional change section.
- Demographic change section.
- State and region summary.
- County appendix.
- Contact and download information.

The prior report works because it does not ask public readers to navigate a
technical package. The 2026 PDF should do the same: tell the story in the PDF,
then point users to the workbook and CSV for exact values.

## What To Update For 2026

The 2026 report should use scenario language throughout. Baseline (Trend
Continuation) can be the main narrative path, but Restricted Growth (CBO
Policy-Adjusted) and High Growth (Elevated Immigration) should be visible near
statewide claims and in the scenario comparison table.

The horizon changes to 2025-2055. The report should print key years for
readability and leave exact annual values to the public downloads.

The methodology page should be plain-language and current. It should describe
the cohort-component method, Census Population Estimates Program inputs,
fertility, mortality, migration, and scenario construction. It should not read
like a technical appendix.

The download language should be simpler than the internal export structure. For
public release, users should see one consolidated Excel workbook and one
consolidated CSV covering the state, regions, and counties.

## What To Avoid

Avoid anything that makes the public PDF feel like a production worksheet:

- Internal file inventories.
- Internal QA controls.
- Methodology decision logs.
- Full annual county tables.
- Full county age-sex tables.
- City/place projections.

The PDF should give a complete public summary, not every available cut of the
data.

## Practical Design Notes

The 2024 report's strongest design lesson is that each section has a clear job.
The statewide page establishes the main trajectory. The caution text explains
why projections should be read carefully. The methodology page gives enough
trust without overwhelming the reader. The regional and county sections make the
statewide number local.

For 2026, keep that rhythm but make the caveats more integrated. Place scenario
and migration caveats near the first statewide chart, not only on the method
page. Keep the county appendix readable by using selected years in the PDF and
sending users to the downloads for annual values.

Marketing will receive the 2024 PDF separately as the visual reference.
