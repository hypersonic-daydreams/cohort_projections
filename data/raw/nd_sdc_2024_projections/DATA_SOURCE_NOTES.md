# ND SDC 2024 Projection Source Notes

## Overview

This directory stores raw North Dakota State Data Center (SDC) 2024 projection
reference files used for methodology comparison, validation, and replication
support.

## Source

- Publisher: North Dakota State Data Center (SDC)
- Project context: SDC 2024 replication and benchmark comparison workflows

## Typical file provenance fields to capture per file

When adding files under this directory, document:
- file name and scenario/year coverage
- source URL or transfer provenance
- download/receipt date (UTC)
- column definitions and any recodes required
- consuming script(s)

## Processing linkage

Used by comparison and review workflows in repository documentation and
supporting scripts.
