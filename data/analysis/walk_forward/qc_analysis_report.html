Key Findings

  • Sensitivity: fertility_rate is the most influential parameter for 2026 Method (state error swing of 4.1 pp). Migration rates dominate uncertainty in both methods.
  • Uncertainty: At 19-year horizons the 95% prediction interval width reaches 0 pp, driven largely by the Bakken boom era. Intervals from recent origins (2015, 2020) are substantially narrower.
  • QC Diagnostics: 7 counties receive D grades (worst-case |error| above 25%): Billings, Dunn, Golden Valley, Grand Forks, McKenzie, Ward, Williams. These are predominantly Bakken oil-patch counties where the boom created unpredictable migration surges.
  • Residuals: Error distributions are non-normal across all horizon buckets for both methods, with moderate positive lag-1 autocorrelation (0.5-0.7). This supports using empirical percentile-based prediction intervals rather than Gaussian assumptions.
