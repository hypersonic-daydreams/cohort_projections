# G02 Outputs

Save ChatGPT outputs here with the following naming convention:

| ChatGPT Output | Save As |
|----------------|---------|
| Full response | `G02_response.md` |
| G02_kpss_resolution.md | `G02_kpss_resolution.md` |
| G02_backtesting_spec.md | `G02_backtesting_spec.md` |
| G02_terminology_corrections.csv | `G02_terminology_corrections.csv` |
| G02_recommendations.md | `G02_recommendations.md` |

## Instructions

1. After ChatGPT completes, copy the full response into `G02_response.md`
2. Download any files ChatGPT produces and save with the G02_ prefix (rename if needed)
3. The G02_terminology_corrections.csv should be a table you can use directly
4. Update the date and status in `G02_response.md`
